Build Secure SPAs with Next Identity

This demo shows you how to easily integrate Next Identity into your single-page applications for secure authentication.

Instructions

  1. Clone the repository: 
    git clone https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPO_NAME.git
  2. Install dependencies: 
    npm install
  3. Configure your Next Identity credentials:
    • Open config.js
    • Replace the placeholder values with your actual client ID, issuer URL, and redirect URI.
  4. Start the development server: 
    serve -s.
  5. Open the app in your browser: Usually at http://localhost:5000
  6. Click "Log In" to authenticate with Next Identity.

Code Example (Authorization)

// app.js
const nextIdentity = {
  authorize: () => {
    //... (code to build authorization URL with PKCE)
    window.location.href = authUrl.toString();
  },
  //... other functions
};

loginButton.addEventListener('click', () => {
  nextIdentity.authorize();
});
